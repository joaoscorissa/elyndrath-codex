# Chronicles of Elyndrath - Project Brief

## Objetivo
Criar um site estático premium (GitHub Pages) inspirado em Diablo IV, Path of Exile 2, Dragon Age Codex e artbooks de dark fantasy.

Tecnologias:
- HTML5
- CSS puro
- JavaScript ES6
- Sem React
- Sem Bootstrap
- Sem Tailwind
- Sem build

O site deve parecer um códice medieval interativo.
